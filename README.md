### Ejecución

```bash
~$ cd program

~$ cp time_program.exe ../
~$ cd ../
~$ ./loop.sh

```
- La carga tiene que ser entre un 1..n, n es la carga máxima que pueda soportar el ordenador.
- Para tener resultado de 1..n, hay que ejecutar el script n veces con diferente carga.
